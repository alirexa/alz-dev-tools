# Git Cheatsheet

- Initialize repo:
  \`\`\`bash
  git init
  \`\`\`

- Add remote:
  \`\`\`bash
  git remote add origin <URL>
  \`\`\`

- Commit and push:
  \`\`\`bash
  git add .
  git commit -m "Initial commit"
  git push -u origin main
  \`\`\`

- Save password with SSH:
  \`\`\`bash
  ssh-copy-id user@server
  \`\`\`
